# 🎰 Emoji Slot Machine

A static web app that turns a 3×3 expression grid into a Facebook-friendly
"slot machine" video. Viewers see the video auto-play in their feed; when they
tap to pause, it lands on a random emoji — like a personality-test meme that
works inside FB.

**Live demo:** https://yazelin.github.io/emoji-slot-machine/

## Features

- Upload a 3×3 grid image (e.g. generated by Gemini / Nano-Banana)
- Automatically splits into 9 individual tiles
- Renders a looping WebM video (Canvas + MediaRecorder, no backend)
- Download or share via the Web Share API

## How it works on Facebook

Facebook auto-plays short videos in the feed and pauses them on the current
frame when the viewer taps. By looping through 9 shuffled emoji frames, the
pause lands on a random one — a free "slot machine" with zero JavaScript on
the viewer side.

## Local dev

No build step. Just open `index.html` in a browser, or:

```
python3 -m http.server 8000
```
